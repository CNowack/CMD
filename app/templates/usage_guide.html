Usage Guide

This page explains how to use each section of the Cancer Microbiome Database. Each analysis page has its own form with specific input requirements documented below.

Shannon Diversity

What it does: Computes per-patient microbial diversity from ASV relative abundances. Results render as an interactive violin plot with individual data points.

How to use:

  1. Choose a cancer type from the dropdown (or All)
  2. Choose a sample type: Buccal, Stool, or Nasal (or All)
  3. Click Add — a chip appears and the violin is added to the plot
  4. Repeat up to 5 combinations to compare side-by-side
  5. Toggle between Shannon Index (H) and Simpson Index (D) with the index dropdown
  6. Click Download Table as CSV to export the raw data

Example queries:

Cancer Type Sample Type Question
Melanoma Stool How diverse are stool microbiomes in melanoma patients?
NSCLC Buccal Is buccal diversity different in lung cancer vs. kidney cancer?
Kidney Carcinoma Buccal (Add alongside NSCLC above to compare)
All Stool What does the global stool diversity distribution look like?
Breast Carcinoma Nasal Any signal in nasal microbiome for breast cancer?

Backend endpoint:

GET /api/cancer_diversity_search?cancer_type=Melanoma&sample_type=stool
GET /api/cancer_diversity_search?cancer_type=&sample_type=stool   ← all cancers
GET /api/cancer_diversity_search?cancer_type=NSCLC&sample_type=   ← all sample types

SIG1/SIG2 Microbiome Signature Search — Prevalence Meta-analysis

What it does: Applies the SIG1/SIG2 framework from Derosa et al. (Cell, 2024) to the BMC cancer microbiome cohort. SIG1 genera are associated with gut dysbiosis and poor immunotherapy response; SIG2 genera are associated with a healthy gut ecosystem and favorable outcomes. Results are shown as genus-level prevalence (proportion of patients with detected relative abundance > 0) across cancer categories or survival groups, side-by-side with the Derosa NSCLC reference cohort.

How to use:

  1. Wait for the genus list to load, then check the genera you want to examine in the SIG1 and/or SIG2 panels (use the search box or Select all to narrow down)
  2. Set analysis parameters:
    • Group by: Cancer Category or Overall Survival (OS<12 vs OS≥12 months)
    • Timepoint: Baseline, Follow Up, Progression, or All
    • Sample type (BMC): Stool, Buccal, Nasal, or All — note the Derosa reference cohort is stool-only
  3. Click Run Analysis — violin plots appear for BMC (left) and Derosa NSCLC (right)
  4. Use the Show radio buttons to display SIG1, SIG2, or both simultaneously
  5. Check or uncheck cancer categories (or OS12 groups) to filter which groups appear on the charts
  6. Toggle Hide genus with zero prevalence in all selected groups to declutter the charts
  7. Click Show data tables under each chart section to expand the raw prevalence numbers
  8. Click CSV next to any table to download that table's data

Example queries:

Genera Group by Sample Type Question
All SIG1 Cancer Category Stool Which dysbiosis-associated genera are most prevalent across cancer types?
All SIG2 Overall Survival Stool Are favorable-response genera more prevalent in OS≥12 patients?
All SIG1 + SIG2 Cancer Category Stool · Baseline How does the BMC cohort compare to the Derosa NSCLC reference across both signatures?
All SIG1 Cancer Category Buccal Is the SIG1 dysbiosis signal also present in buccal samples?

Important caveats: The BMC cohort used 16S rRNA amplicon sequencing; the Derosa cohort used shotgun metagenomics. Detection sensitivity and taxonomic resolution differ between platforms — cross-cohort comparisons should be interpreted with caution. This analysis is purely exploratory and hypothesis-generating.

Taxonomic Sankey

What it does: Visualizes the taxonomic composition of microbiome samples as an interactive Sankey (flow) diagram. Abundance flows left to right through five taxonomic ranks: Phylum → Class → Order → Family → Genus. The width of each flow is proportional to the number of observations at that rank. Filtering by sample ID additionally shows the Genus level label.

How to use:

  1. Choose a Filter by type from the first dropdown:
    • Sample ID — a single sample; shows the full five-rank hierarchy down to Genus
    • Cancer type — aggregates all samples from patients with that cancer type
    • Treatment — aggregates all samples from patients on that treatment
    • Sample type — aggregates by collection method (Buccal, Stool, or Nasal)
  2. Choose a specific Value from the second dropdown (populated automatically based on filter type)
  3. Click Render Sankey — the diagram draws below with rank labels along the bottom
  4. Hover over any flow or node to see counts

Example queries:

Filter by Value Question
Sample type Stool What is the overall taxonomic structure of all stool samples in the cohort?
Cancer type Melanoma Which phyla and families dominate the microbiome of melanoma patients?
Cancer type NSCLC How does NSCLC taxonomic composition compare to Melanoma?
Sample type Buccal Does buccal microbiome composition differ structurally from stool?
Sample ID Any single SID What is the full genus-level taxonomy of one specific sample?

Backend endpoint:

GET /api/sankey_data?filter_type=sample_type&filter_value=stool
GET /api/sankey_data?filter_type=cancer_type&filter_value=Melanoma
GET /api/sankey_data?filter_type=sid&filter_value=<sample_id>
