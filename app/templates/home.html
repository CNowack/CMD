Welcome to the Cancer Microbiome Database

About this project

The Cancer Microbiome Database (CMD) is a relational database and interactive web application that integrates 16S rRNA microbiome sequencing data with clinical patient metadata from Boston Medical Center (BMC). It provides a unified, publicly accessible resource to explore microbiome composition, visualize phylogenetic relationships, and track longitudinal shifts in patients' microbiome profiles across cancer treatment timepoints.

The database contains data from 547 samples collected from 142 BMC cancer patients across three anatomical sites (buccal, stool, nasal) over multiple timepoints since 2019.
